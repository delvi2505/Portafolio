Manejador de cuentas de usuario

Este sistema de control de cuentas de usuario tiene la finalidad de servir como código base para otros proyectos del mismo entorno que requieran el manejo de cuentas de usuarios y permisos de los mismos dentro del sistema.

Entorno de ejecución: Cliente – Servidor.

Patrón de diseño: MVC (Modelo Vista Controlador).

Tecnologías de construcción y diseño:
  • Html
  • CSS
  • Bootstrap
  • Jquery
  • Motor de plantillas Razor
  • Lenguaje de programación C#
  • .Net Framework
  • Entityframework
  • SQLServer
