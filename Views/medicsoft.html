MedicSoft

MedicSoft es un sistema informático multi-usuarios concebido bajo la idea de apoyar las funciones del área de la salud, automatizando procesos, digitalizando los datos, mejorando la confiabilidad de la información y facilitando su comprensión, análisis y consulta.

Entorno de ejecución: Cliente – Servidor (Back-end sobre Windows).

Patrón de diseño: MVC (Modelo Vista Controlador).

Tecnologías de construcción y diseño:
  • Html
  • CSS
  • Bootstrap
  • JavaScript
  • Jquery
  • Motor de plantillas Razor
  • Lenguaje de programación C#
  • .Net Framework
  • Entityframework
  • MariaDB
