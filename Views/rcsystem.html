RC-System

RC-System es un sistema informático de escritorio multi-usuarios dirigidos a empresas de alquiler de autos el cual les permite automatizar sus procesos y llevar un mejor control y organización de su actividad.

Entorno de ejecución: Cliente – Servidor.

Patrón de diseño: MVC (Modelo Vista Controlador).

Tecnologías de construcción y diseño:
  • Windows Forms
  • Lenguaje de programación C#
  • .Net Framework
  • MariaDB
